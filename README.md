<h1>Women Safety Analytics</h1>

<p>
  Women Safety Analytics is a simple AI-based system that detects people, identifies gender, 
  and finds situations that may be unsafe for women. It gives quick alerts and shows 
  easy visualizations to help improve safety in public areas.
</p>

<h2> Key Features</h2>
<ul>
  <li>Person detection from live camera feed</li>
  <li>Gender classification (male/female)</li>
  <li>Gender distribution count</li>
  <li>Lone woman detection at night</li>
  <li>Woman surrounded by men detection</li>
  <li>SOS gesture detection (basic)</li>
  <li>Simple alert system</li>
  <li>Hotspot identification from past alerts</li>
  <li>User-friendly interface for women</li>
  <li>Easy charts and visuals for understanding data</li>
</ul>

<h2> Tech Used</h2>
<ul>
  <li>Python, OpenCV</li>
  <li>Deep Learning (TensorFlow/Keras)</li>
  <li>Firebase Database</li>
  <li>HTML, CSS, JavaScript (Frontend)</li>
</ul>

<h2> How It Works</h2>
<ol>
  <li>Camera feed is processed frame by frame.</li>
  <li>AI model detects people and classifies gender.</li>
  <li>System checks for unsafe patterns.</li>
  <li>If found, an alert is created and shown on dashboard.</li>
  <li>Visuals help understand trends and hotspot areas.</li>
</ol>
